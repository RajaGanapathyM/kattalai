## How to Get Listed

1. Read [`CONTRIBUTOR_LICENSE_AGREEMENT.md`](./CONTRIBUTOR_LICENSE_AGREEMENT.md).
2. Open a pull request with your contribution.
3. Add the following statement in your PR description:

   > *"I have read the CONTRIBUTOR_LICENSE_AGREEMENT.md and I hereby sign it for all my contributions to this project."*

4. The maintainer will add your entry to this file when the PR is merged.
