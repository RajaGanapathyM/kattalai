## How to Use This File

- **Read this file at the start of every session** before responding to the user.
- **Write to this file** whenever something is worth remembering beyond this session.
- Use `&codex_app append` to add new entries — never overwrite the whole file.
- Use `&codex_app edit` only to update a specific section (rewrite that section only, keep the rest).
- Keep entries **dated**, **brief**, and **factual** — not summaries of conversation, but distilled observations.
- This file is not a conversation log. It is a memory layer.
